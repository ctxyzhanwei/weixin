认证上网设置 (用户关注认证后可以上网，之后再次来店可免认证，取消关注后需要重新认证)
*上网关键字：
户输入此关键词将会返回认证提示，多个关键词之间使用空格分隔。例如：wifi 上网 免费上网
*回复文字：
输入关键词后的返回认证文字。例如：10分钟内点击此文字，认证后可以上网x分钟
*获取验证码关键词：
用户输入此关键词将会返回上网验证码。例如: 电脑上网 上网验证码
节点名称： {pigcms:$node_info.wx_id}
创建时间： {pigcms:$node_info.created_at}
更新时间： {pigcms:$node_info.updated_at}
*图片回复链接：
将此链接填入图片回复的"URL或活动外链代码"即可完成图片回复认证
*认证页地址：
输入认证页面URL，以http://开头，例如：http://demo.rippletek.com/ext/portal/rptk3wxandcode.html
*认证成功地址：   
输入成功页面URL，以http://开头，例如：http://demo.rippletek.com/ext/portal/rptksuccess.html
*开启认证： checked="checked" >
*未认证可微信聊天：   未认证可使用微信聊天的时间，无法在微信中访问网页。不填写为可以使用微信聊天10分钟
*认证后可上网：   认证成功后可以上网时间，不填写为可以上网1440分钟
*再次认证方式：   "简单认证"即在二次连接时候，需要打开浏览器点击任意网页。选择"与首次一致"，无法实现取消关注后需要重新认证。
*IOS关闭主动弹窗： checked="checked" >  IOS终端(iPhone, iPod, iPad)和WP终端将不会主动弹窗，和Android认证流程一致
*网址白名单：
使用英文逗号(,)分隔主机名或IP地址，例如：qq.com,weibo.co
