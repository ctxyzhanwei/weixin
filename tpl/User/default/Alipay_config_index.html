鉴别微信支付新老版本的三种方法：
1. 9月中旬后申请的微信支付应该为新版，
2. 申请微信支付时发来两个通知邮件是老版，
3. 申请微信支付时微信发来的邮件中带有“paysignKey”字样是老版！

因为微信端的新版微信支付错误提示不完整且不易读，
测试新版微信支付时，如果出现任何问题请先认真检查配置项是不是完全填写正确。
